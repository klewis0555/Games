# Overview
Mastermind is a game that is also sometimes called bulls and cows.

Clean this up later
## Basics
- 4 digit code
- some use numbers, some use colors
- some allow repeats or blanks
- code breaker has a certain number of guesses
- each guess gets a result showing how correct they were
  - a correct number in the correct position is a bull
  - a correct number in the wrong position is a cow
- keep guessing until either they crack the code or they run out of guesses
